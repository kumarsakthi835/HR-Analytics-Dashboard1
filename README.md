# HR Analytics Dashboard (Power BI)

## Overview

This project presents an HR Analytics Dashboard built using Power BI to analyze employee data and generate actionable insights. It helps in understanding workforce trends, employee behavior, and key factors influencing attrition.

## Objectives

* Analyze employee attrition and retention
* Understand workforce demographics
* Evaluate job satisfaction and performance
* Support data-driven HR decisions

## Dataset

The dataset includes:

* Employee details (age, gender, education)
* Job role and department
* Salary and tenure
* Job satisfaction and performance rating
* Attrition status

## Features

* Attrition analysis by department and role
* Employee demographic insights
* Salary and attrition comparison
* Job satisfaction and performance analysis
* Interactive filters for dynamic exploration

## Tools Used

* Power BI
* DAX
* Excel / CSV

## Key Insights

* Higher attrition in certain roles and departments
* Low job satisfaction linked to higher turnover
* Employees with shorter tenure are more likely to leave
